# Esperanto Cheat Sheet – Quick Reference

## Personal Pronouns

| Singular | Meaning | Plural | Meaning |
|----------|---------|--------|---------|
| mi | I | ni | we |
| vi | you (sing.) | vi | you (pl.) |
| li | he | ili | they |
| ŝi | she | | |
| ĝi | it | | |

*Note: `vi` is used for both singular formal and plural informal/formal.*

---

## Present Tense Verbs (-as ending)

**Infinitive form → Present tense form**

lerni (to learn) → mi lernas (I learn)
labori (to work) → vi laboras (you work)
manĝi (to eat) → li manĝas (he eats)
dormi (to sleep) → ŝi dorsmas (she sleeps)
studi (to study) → ni studas (we study)

text

**Rule:** Infinitive (ending in -i) → remove -i, add -as

---

## Accusative Case (-n ending) for Direct Objects

Nominative → Accusative

pomo (apple) → pomon (apple as object)
hundo (dog) → hundon (dog as object)
letero (letter) → leteron (letter as object)

Example sentences:
✓ Mi manĝas pomon. (I eat an apple.)
✓ Vi vidas hundon. (You see a dog.)
✓ Ŝi skribas leteron. (She writes a letter.)

❌ Mi manĝas pomo. (WRONG – missing -n)
❌ Vi vidas hundo. (WRONG – missing -n)

## Common Errors & Fixes

### ❌ Missing -n on Direct Objects

WRONG: Mi vidas pomo.
RIGHT: Mi vidas pomon.
REASON: "Pomo" is a direct object → needs -n

text

### ❌ Wrong Pronoun Gender

WRONG: Ŝo laboras. / Lo laboras. (using wrong letters)
RIGHT: Ŝi laboras. / Li laboras.
REASON: ŝi = she, li = he. These forms are fixed.

text

### ❌ Wrong Verb Ending

WRONG: Mi labor. / Vi labor.
RIGHT: Mi laboras. / Vi laboras.
REASON: Present tense ALWAYS ends in -as

text

### ❌ Confusing Infinitive and Present

WRONG: Mi lerni.
RIGHT: Mi lernas.
REASON: Infinitive = lerni. Present = lernas.
